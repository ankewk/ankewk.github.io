mysql利用存储过程生成千万级别数据

随着程序数据量越来越大，sql优化越来越重要。
这里就讲一下如何去mysql调优的生成千万级别的测试数据。

##1.创建学生表stu
先用MYISAM存储引擎，可以快色的插入数据。

delimiter //
CREATE PROCEDURE insert_stu_p (IN num INT)
BEGIN
DECLARE n INT DEFAULT 1;
WHILE n <= num DO
INSERT INTO stu (name, age) values (concat(‘wk’,n), rand()*20);
SET n=n+1;
END WHILE;
END
//

CALL insert_stu_p(10000000);

ALTER TABLE stu ENGINE=INNODB;
```
大量数据生成，就可以开始mysql优化方面的学习了。
